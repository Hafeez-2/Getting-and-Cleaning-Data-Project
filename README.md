# Getting and Cleaning Data Course Project

This repository contains the files for the Coursera
Getting and Cleaning Data course project.

## Files
- run_analysis.R: Script to clean and process the data
- tidy_data.txt: Final tidy data set
- CodeBook.md: Description of variables and transformations

## Instructions
Place the UCI HAR Dataset folder in your working directory
and run run_analysis.R to generate tidy_data.txt.
